# 1 笔记总结
1. 编程对比机器学习
	- 传统编程：显式编程，根据人解决问题的经验
	- 机器学习：非显式编程，人类编写机器学习算法，从数据中训练机器学习模型(程序)
2. 机器学习优势
	- 基于大量数据：网络、新闻
	- 输出个性化：新闻、物品、广告推荐
	- 人类不能解释专业知识：语音、人脸识别
	- 专业知识不存在
3. 机器学习类型
	- 预测：监督学习、无监督学习(生成式问题)
	- 决策：强化学习，转变到新的状态
	- 区别：是否直接改变环境
4. 机器学习三大类问题
	- 预测类任务：监督学习
	- 生成类任务：无监督学习
	- 决策类任务：强化学习
5. 机器学习应用
	- 推荐广告
	- 人脸识别
	- 医疗图像分析
	- 工业检修
	- 工业节能
	- 图像/音乐/视频生成
	- 机器人控制
