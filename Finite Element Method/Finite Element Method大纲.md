## 10.4 变分原理
类比最优化理论和变分原理，二者之间有一些关联，针对约束的处理也有相似之处，例如都会采用拉格朗日乘子法、增广拉格朗日乘子法、罚函数法等方法将有约束问题转化成无约束问题。
其次，最优化理论是寻找损失函数的最值，损失函数是实数空间到实数的映射，即在实数空间中寻找最优解；变分原理是寻找泛函的最值，泛函是函数空间到实数的映射，即在函数空间中找一个函数，使得泛函取得极值，从这个角度看，最优化理论和变分原理同属一大类问题。

| 名称    | 优化目标 | 映射关系       | 约束        | 备注                                |
| ----- | ---- | ---------- | --------- | --------------------------------- |
| 最优化理论 | 损失函数 | 实数空间 -> 实数 | 变量需要满足的条件 |                                   |
| 变分原理  | 泛函   | 函数空间 -> 实数 | 函数之间的约束关系 | 对于泛函的极值问题，需要定义函数空间的距离、连续性等，相对比较抽象 |
